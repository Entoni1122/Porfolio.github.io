Jams and Projects

Below you can find a list of my personal and school projects, as well as jams I have participated in.
Wild Clone West
"It's a school project made in Unity for our final second-year exam. It was completed in one month with my academy friends. In this project, I was in charge of the UI, player movement, shooting system, and level design."
Wachy Wheels
"Wachy is a personal project developed with a friend, designed for mobile. I was responsible for the UI, pedestrian movement, VFX, and power-up implementation."
Airport Delivery Guy
"ADG was created for the Xplace Game Jam. I was responsible for player movement, stacking mechanic, vehicle implementation, and UI design."
Battle Slot
"Battle Slot is a game created for the 2024 Global Game Jam. We secured second place in our school's tournament with this project. I was responsible for the slot machine functionality and in-game actions."
Coffe break
"Coffee Break" is an upcoming game on Steam, developed for a game jam that we won. Schools across Italy were challenged to create a functional game based on the theme "space". I contributed by developing all the mechanics of the coffee machine, designing the radio system that changes music based on the rotation of the crank, and implementing various in-game actions.
